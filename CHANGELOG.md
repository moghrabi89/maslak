# سجل التغييرات

## غير منشور

- إضافة المراجعة المتباعدة الذكية ومركز `/review`.
- إضافة نظام الشارات العلمية وخدمة منح الشارات.
- ربط الشارات بإنهاء التحديات والمراجعات.
- تحويل `/leaderboard` إلى صدارة حقيقية مبنية على بيانات المستخدمين.
- تحويل `/shop` و`/profile` إلى شاشات مبنية على بيانات فعلية.
- إضافة لوحة `/admin/analytics` لتحليل التعثر دون كشف بيانات الطلاب الحساسة.
- إضافة سياسة توسعة المحتوى ووثيقة التشغيل وخارطة المنهج المستقبلية.
- إضافة اختبارات لقواعد المراجعة المتباعدة وقواعد الشارات.

## ملاحظات اعتماد

- لا تضيف هذه التغييرات أي محتوى فقهي منشور جديد خارج نطاق MVP.
- توسعة سفينة النجاة وبقية الكتب موثقة كمسار draft/review/publish فقط.
- يجب تنظيف أخطاء lint القديمة قبل الإطلاق العام.
